# Order can be completed with an empty cart

**Severity:** Major

**Priority:** Medium

**Environment:** Brave v1.92.140

**Device:** PC

**OS:** Windows 11

**Preconditions:**

User is logged in.
Shopping cart is empty.

**Steps to reproduce:**

1. Open the shopping cart.
2. Click Checkout.
3. Enter valid checkout information.
4. Click Continue.
5. Click Finish.

**Expected result:**

The application prevents checkout completion when the cart is empty and displays an appropriate message.

**Actual result:**

The order is completed successfully with an empty cart.

**Frequency:**

Always.
