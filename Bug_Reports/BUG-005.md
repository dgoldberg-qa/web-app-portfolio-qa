# Last name value replaces First name during checkout

**Severity:** Major

**Priority:** Medium

**Environment:** Brave v1.92.140

**Device:** PC
**OS:** Windows 11

**Preconditions:**

User is logged in as problem_user.
At least one product has been added to the cart.

**Steps to reproduce:**

1. Open the shopping cart.
2. Click Checkout.
3. Enter John in the First name field.
4. Enter Doe in the Last name field.

**Expected result:**

The checkout form should display the entered values correctly:

First name: John
Last name: Doe

**Actual result:**

The value entered in the Last name field replaces the First name value.

**Frequency:**

Always.
