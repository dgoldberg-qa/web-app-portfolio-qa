# Incorrect product images are displayed

**Severity:** Minor

**Priority:** Low

**Environment:** Brave v1.92.140

**Device:** PC

**OS:** Windows 11

**Preconditions:**

User is logged in as problem_user.

**Steps to reproduce:**

Open the Products page.
Compare product names with displayed images.

**Expected result:**

Each product should display the corresponding image.

**Actual result:**

The products display incorrect image that does not match the product names.

**Frequency:**

Always.
