# Cart contents are shared between different user accounts

**Severity:** Major

**Priority:** Medium

**Environment:** Brave v1.92.140

**Device:** PC

**OS:** Windows 11

**Preconditions:**

User has access to two registered accounts.
User is logged in as standard_user.

**Steps to reproduce:**

1. Log in as standard_user.
2. Add any product to the cart.
3. Log out.
4. Log in with another account.
5. Open the shopping cart.

**Expected result:**

The cart should contain only products added by the currently logged in user.

**Actual result:**

Products added by another user account are displayed in the cart.

**Frequency:**

Always.
